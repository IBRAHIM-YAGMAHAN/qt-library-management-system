# Qt Library Management System

A desktop library automation app built with Qt (C++) and SQLite. 
Manages library members, books, and book borrowing/return operations.

## Features
- Member management (add, update, delete, search)
- Book management with stock tracking
- Borrow books with date tracking
- Return books with automatic late fee calculation (4 TL/day after 15 days)
- Prevents deletion of members with unreturned books
- Prevents borrowing when stock is 0

## Setup
1. Open the project in Qt Creator
2. Build and run
3. Database (SQLite) is created automatically on first launch
